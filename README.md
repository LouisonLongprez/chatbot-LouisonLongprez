## Installation
Dans VSC, ouvrir le terminal dans le dossier chatbotlouison et remplir avec ces commandes :
yarn add
yarn init

yarn add axios
yarn add discord.js
yarn add openai
yarn add dotenv
yarn add index.js

yarn run
